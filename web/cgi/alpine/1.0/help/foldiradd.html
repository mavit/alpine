Creating Folders

Folders are used to contain messages. Directories are used to contain folders.

After you click on the Folder List page, you can create either one.

  • To add a folder: enter the name of the folder you want to add to in the "New folder name:" text box. Click .
  • To add a directory: enter the name of the directory you want to add to in the "Create New Directory:" text box. Click .

Click to return to the Folder List screen without creating a new directory or folder.
